# Treck Motors Cuba — Frontend

Cliente React (Vite) para Treck Motors Cuba.

## Desarrollo

```bash
npm install
npm run dev        # Vite dev server en :5173 (proxy /api → localhost:3000)
```

## Build

```bash
npm run build      # Genera SPA en dist/
```

Revisa el README raíz para instrucciones completas de deploy y configuración de Supabase.
